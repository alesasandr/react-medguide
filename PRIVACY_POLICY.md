# Политика конфиденциальности MedGuide

Дата последнего обновления: 18.12.2025

## 1. Общие положения
MedGuide — мобильное приложение для поиска информации о лекарственных препаратах и ведения истории выдачи препаратов пользователем (врачом/сотрудником).

Настоящая Политика конфиденциальности описывает, какие данные мы собираем, как мы их используем и какие у вас есть права.

## 2. Какие данные мы собираем
- **Данные учётной записи**: e-mail, пароль (пароль хранится на сервере в виде хэша).
- **Профиль**: ФИО/имя, специализация, место работы, табельный/служебный ID, ссылка на аватар (если указана).
- **История выдачи препаратов**: препарат, количество, дата/время выдачи, идентификатор/имя врача (если есть).
- **Технические данные**: диагностические данные об ошибках (например, код ответа API) — для улучшения стабильности (в debug режиме).

## 3. Как мы используем данные
- для регистрации и входа в приложение;
- для отображения профиля пользователя;
- для выполнения функций приложения (просмотр препаратов, выдача, история выдачи);
- для обеспечения безопасности и предотвращения злоупотреблений;
- для улучшения качества работы приложения.

## 4. Хранение и безопасность
- Данные профиля и история выдачи могут сохраняться **локально на устройстве** (кеш/офлайн-режим).
- Сервер хранит данные в базе данных (PostgreSQL).
- Для авторизации используется токен (DRF Token Authentication).

Мы принимаем разумные меры защиты данных, однако не можем гарантировать абсолютную безопасность при передаче данных через интернет.

## 5. Передача данных третьим лицам
Мы не продаём и не передаём персональные данные третьим лицам, за исключением случаев:
- если это требуется законом;
- если это необходимо для предоставления сервиса (например, почтовый провайдер для восстановления пароля).

## 6. Ваши права
Вы можете:
- обновить данные профиля в приложении;
- запросить удаление/изменение данных, написав нам (см. контакты).
