# MedGuide — метаданные для публикации (Google Play / App Store)

## 1) Название приложения
**MedGuide**

## 2) Короткое описание (до ~80 символов)
Поиск препаратов, карточки, выдача и история. Работает с серверной базой.

## 3) Полное описание
MedGuide — приложение для быстрого поиска лекарственных препаратов, просмотра карточек и ведения истории выдачи.

Возможности:
- список препаратов и карточка препарата (форма, дозировка, артикул, QR);
- поиск по артикулу;
- сканирование QR‑кода препарата;
- выдача препарата с обновлением остатка;
- история выданных препаратов;
- профиль пользователя;
- восстановление пароля по e‑mail.

## 4) Ключевые слова (пример)
медицина, лекарства, препараты, склад, учет, аптечка, QR, артикул, врач, клиника

## 5) Категория
**Медицина** / **Здоровье**

## 6) Возрастной рейтинг (рекомендация)
**3+** (без контента для взрослых)

## 7) Политика конфиденциальности

## 8) Скриншоты (чек‑лист)
Сделать и загрузить минимум 4–6 скриншотов:
- Экран входа
- Главный экран (кнопки)
- Список препаратов
- Карточка препарата + QR
- Экран выдачи/кнопка “Взять препарат”
- Профиль + история выдачи
